# Актуальные вакансии (формат и пример)

Этот файл — образец того, как добавлять в базу знаний актуальные вакансии
(например, выгрузку с hh.kz / enbek.kz). Формат описан в README. Добавляйте по
одному markdown-файлу на выгрузку или по вакансии — после добавления запускайте
переиндексацию (`python ingest.py` или POST /admin/ingest).

## Пример вакансии

**Должность:** Тренер по плаванию
**Организация:** ДЮСШ №3, г. Астана
**Источник:** hh.kz (вакансия № 0000000), дата публикации: 2026-06-01
**Зарплата:** по результатам собеседования
**Занятость:** полная

**Обязанности:**
- Проведение тренировок групп начальной подготовки.
- Подготовка спортсменов к городским и областным соревнованиям.
- Ведение учебной документации.

**Требования:**
- Высшее физкультурное образование.
- Спортивный разряд по плаванию приветствуется.
- Опыт работы с детьми.

**Контакты:** hr@example.kz
